🚀 Xcelerate - Modern Meme Token Launchpad on X Layer Project Overview Buatkan Xcelerate, sebuah platform launchpad meme token modern yang dibangun di atas blockchain X Layer (OKX zkEVM L2). Platform ini mengadopsi mekanisme bonding curve dan graduasi otomatis ke DEX, dengan arsitektur smart contract modular yang terdiri dari 8 kontrak terpisah. 🎯 Core Features Smart Contract Architecture (8 Contracts) Registry - Hub koordinasi pusat untuk semua kontrak TokenFactory - Membuat token ERC20 meme baru BondingCurveContract - Mengelola harga token dengan virtual reserves LiquidityPoolContract - Menangani manajemen likuiditas dan swap UserManagement - Kontrol akses dan perlindungan spam FeeManager - Pengumpulan fee dan distribusi reward MarketGraduation - Graduasi token ke QuickSwap DEX WOKB - Wrapped OKB untuk kompatibilitas DEX Frontend Requirements Framework: React + Vite (untuk Vercel deployment yang optimal) Styling: Tailwind CSS dengan glass-morphism design Web3: Wagmi + Viem untuk interaksi blockchain UI/UX: Modern, responsive, dengan animasi smooth Deployment: Vercel-ready dengan konfigurasi Next.js atau Vite 💰 Tokenomics & Economics Pricing Mechanism Virtual Reserves: 30 OKB, 1.073B tokens Graduation Threshold: 80 OKB collected Base Liquidity Lock: 36 OKB permanently locked Creation Fee: 0.001 OKB per token Fee Distribution Platform: 2% - Pengembangan protokol Creator: 1% - Reward untuk pembuat token Stakers: 1% - Reward staking platform LPs: 0.5% - Insentif liquidity provider Referrers: 0.5% - Program referral
